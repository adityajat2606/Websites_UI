# 🚀 Project Setup Guide

This project is a modern frontend application built with a fast
development workflow and live reloading for an efficient development
experience.

------------------------------------------------------------------------

## 📦 Installation

Follow the steps below to set up the project locally.

### 1️⃣ Clone the Repository

git clone <your-repository-url>


### 2️⃣ Navigate to the Project Folder

cd <project-folder-name>

### 3️⃣ Install Dependencies

Install all required packages using npm.

npm install


### 4️⃣ Start the Development Server

Run the development server with hot reload enabled.

npm run dev


After running this command, the application will start and be available
at:

    http://localhost:3000

The server automatically reloads whenever you make changes to the source
files.
